getAddress 

- function getAddress (uint ind) external view returns( address) 
- inputs 
    0: factory
    x: vault address

    ie:
    1: weth/dai 0.05%
    2: weth/usdc 0.3%
    ....

- returns
    vault address 
        or 
    address(0): invalid input number
